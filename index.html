Introducing Cloud Cannon

What is it?

Cloud Cannon provides simple Hosting with a CMS for Web Designers. There's no programming required, files are easily synced using Dropbox and it takes just 2 minutes to setup.

How does it work?

Sync

Simply copy your HTML, CSS and other website files into Dropbox to automatically sync them to Cloud Cannon.

Client Access

Add class="editable" to the elements in the HTML you want your client to update.

Live Website

You now have a blazing fast live website with a CMS your clients can use to update their content.

For more details check out Cloud Cannon.
